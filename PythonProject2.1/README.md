# ЭУ Аналитика (Django)

## Что делает приложение

Приложение предназначено для **аналитики учебных результатов**:

- загрузка результатов обучения из CSV (для роли "Руководитель кафедры"/администратора);
- сводная панель (фильтры по семестру/дисциплине/преподавателю);
- детализация по группам и дисциплинам;
- экспорт данных в CSV и отчёта в PDF;
- разграничение доступа по ролям (преподаватель видит только «свою» аналитику).

## Быстрый старт

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Открыть: `http://127.0.0.1:8000/`

## Учётные записи для проверки

Миграция `analytics.0004_seed_roles_and_demo_users` создаёт демонстрационные роли и пользователей:

- **Преподаватель**: `teacher` / `demo12345`
- **Руководитель кафедры**: `head` / `demo12345`

При необходимости пароль можно изменить в миграции или создать пользователей вручную в админке.

## Важно про доступ

- Вход: `/login/`
- Выход: `/logout/` (кнопка в шапке)
- Панель: `/dashboard/`
- Загрузка CSV: `/upload/` (кнопка "Загрузка CSV" видна только руководителю)

CSV-формат (разделитель `;`):

Колонки: `group;student;discipline;teacher;year;term;grade;attendance`
